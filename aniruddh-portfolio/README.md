# Aniruddh Murasiya - Electrical Engineering Portfolio

A visually polished GitHub Pages portfolio for graduate/junior electrical engineering applications.

## Files
- `index.html` - website structure and portfolio content
- `style.css` - modern visual design, responsive layout and animations
- `script.js` - mobile menu and scroll reveal animations
- `assets/Aniruddh_Murasiya_CV.pdf` - CV linked from the hero section

## Before uploading
1. Replace `https://github.com/YOUR-GITHUB-USERNAME` in `index.html` with your real GitHub URL.
2. Replace placeholder visuals with your screenshots when ready:
   - AutoCAD electrical layout
   - LabVIEW / NI DAQ project screenshot
   - MATLAB/Simulink HVDC model or response graph
   - Transformer testing workflow/checklist
3. Keep any company documents/photos confidential. Use recreated diagrams instead.

## Upload to GitHub Pages
1. Create a new GitHub repository called `aniruddh-portfolio`.
2. Upload all files and folders from this package.
3. Go to Settings → Pages.
4. Under Source, choose `Deploy from a branch`.
5. Select `main` branch and `/root` folder.
6. Save and wait for GitHub to publish your site.

For a cleaner personal website, create the repository as:
`YOUR-GITHUB-USERNAME.github.io`
